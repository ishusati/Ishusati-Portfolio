# Ishu Sati — iOS Developer Portfolio v2.0

Premium Apple-inspired dark portfolio. Built with React + Vite.

## ⚡ Quick Start (3 steps)

```bash
# 1. Open this folder in terminal
cd ishu-portfolio

# 2. Install
npm install

# 3. Run
npm run dev
```

Open → **http://localhost:5173**

## 🚀 Build for Production

```bash
npm run build
npm run preview
```

## 📁 Structure

```
src/
├── components/
│   ├── Navbar.jsx      ← Pill-style sticky nav
│   ├── Hero.jsx        ← Name + circular photo + stats
│   ├── About.jsx       ← 3 highlight cards + education
│   ├── Experience.jsx  ← Animated timeline, 5 companies
│   ├── Skills.jsx      ← 2×2 category cards with tags
│   ├── Projects.jsx    ← 8 projects, all App Store links live
│   ├── Contact.jsx     ← 2×2 contact grid + copy email
│   └── Footer.jsx
├── useReveal.js        ← Shared scroll animation hook
├── App.jsx
├── main.jsx
└── index.css
```

## ✅ Features
- Exact dark blue theme from your screenshots
- Pill navbar with active section tracking
- No rotating profile photo — subtle float animation only
- All 8 App Store links working and correct
- Professional content from your resume
- Scroll-reveal animations on every section
- Fully responsive (mobile-first)
- Zero external dependencies beyond React
